# Show Package Citation

Display the recommended citation for scMetaLink

## Usage

``` r
citationScMetaLink()
```

## Value

Invisibly returns NULL, prints citation information

## Examples

``` r
citationScMetaLink()
#>

#>   communication from single-cell transcriptomic data.
```
