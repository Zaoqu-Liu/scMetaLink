# Get Top Sensing Cell Types for a Metabolite

Get Top Sensing Cell Types for a Metabolite

## Usage

``` r
getTopSensors(object, metabolite, top_n = 5)
```

## Arguments

- object:

  scMetaLink object

- metabolite:

  Character. Metabolite HMDB ID or name

- top_n:

  Integer. Number of top cell types to return

## Value

data.frame with top sensing cell types
